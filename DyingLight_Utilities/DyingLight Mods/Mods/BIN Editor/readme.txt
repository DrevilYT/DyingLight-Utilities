= Instruction =
1. Drag the bin file you want to edit on the exe
(common_texts_all.bin)
2. Txt file will appear, edit everything in it to your liking.
(common_texts_all.bin.txt)
3. Drag the original bin file onto the exe again.
(common_texts_all.bin -> drag -> DyingLightBIN.exe)
4. bin2 file will appear, delete the original bin file and rename the bin2 extension to bin
(common_texts_all.bin -> delete)
(common_texts_all.bin2 -> rename -> common_texts_all.bin)
